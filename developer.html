 Beta

Para dispositivos com iOS 16.4 ou posterior:

Acesse Ajustes > Geral > Atualização de Software > Atualizações Beta e escolha a versão beta do iOS que deseja instalar.

Abra as Atualizações Beta

Nota: para acessar o software beta, seu iPhone deve ter uma sessão iniciada com a mesma Conta Apple que você usou para se registrar no Programa Apple Beta Software.

Para dispositivos com iOS 16.3 ou anterior:

No seu iPhone, baixe o perfil de configuração. O iPhone notificará você quando o perfil for baixado. Abra os Ajustes, toque no perfil e siga as instruções na tela para permitir a instalação.

③ Atualize seu dispositivo iOS

Você pode instalar a versão beta do iOS pela Atualização de Software, na seção Geral dos Ajustes. Se uma versão beta não estiver disponível, você receberá uma notificação quando ela for lançada.
